# gender-sensitivity-code
Replication code for working paper on gender differentials in agricultural sensitivity to extreme heat, containing stata do-files necessary to reproduce the findings of our research: 

## Regression
- "Ethiopia_regression.do" "Nigeria_regression.do": Stata scripts used to run within-country panel regressions for estimating gendered impacts of extreme heat on smallholder agriculture in Ethiopia and Nigeria, respectively. Outputs are written out as figures or tables in Main Text and Supplementary Information.

- "SSA_regression.do": Stata scripts used to run cross-sectional regressions for estimating gendered impacts of extreme heat on smallholder agriculture across 7 SSA countries. 

## Cleaning and Harmonizing
- The "\Data cleaning" directory includes 7 do-files used for cleaning and harmonizing nationwide survey data from Ethiopia, Nigeria, Malawi, Mali, Niger, Tanzania, and Uganda.
